# diabetes

糖尿病预治智能助手项目仓库。

当前仓库根目录按子项目组织，便于后续继续补充后端与部署内容。

## 目录结构

- `frontend/`
  Vue 3 移动端前端项目，当前已完成原型页面、路由、风险评估交互和 Dify Chatflow 流式接口预留。

## 后续规划

- `express/`
  后续用于放置 Express 鉴权、数据库访问、Dify 代理与业务编排服务。

## 说明

- `frontend/node_modules/` 与 `frontend/dist/` 不提交到仓库。
- 设计文档与课程过程文件保留在本地工作目录，不作为当前 Git 仓库内容提交。
